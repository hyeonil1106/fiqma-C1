# 이미지 폴더

이 폴더에는 삼성물산 패션부문 랜딩 페이지에서 사용하는 이미지 파일들이 위치합니다.

## 필요한 이미지 파일 목록

이미지를 다운로드하려면 프로젝트 루트의 `IMAGE_DOWNLOAD_GUIDE.md` 파일을 참고하세요.

### 파일 목록:
- hero-bg.jpg (히어로 배경 이미지)
- hero-overlay.jpg (히어로 오버레이 이미지)
- news-1.jpg (뉴스룸 이미지 1)
- news-2.jpg (뉴스룸 이미지 2)
- news-3.jpg (뉴스룸 이미지 3)
- news-4.jpg (뉴스룸 이미지 4)
- brand-1.jpg (브랜드 이미지 1)
- brand-2.jpg (브랜드 이미지 2)
- brand-3.jpg (브랜드 이미지 3)
- career-bg.jpg (커리어 배경 이미지)

## 이미지 추가 방법

1. 프로젝트 루트의 `IMAGE_DOWNLOAD_GUIDE.md` 파일을 열어 다운로드 가이드를 확인합니다.
2. 제공된 curl 명령어를 실행하거나 다운로드 링크를 클릭하여 이미지를 다운로드합니다.
3. 다운로드한 이미지를 이 폴더(`public/images/`)에 저장합니다.
